Исследование влияния параметров на ЦФ проводилось на случайно сгенерированной задаче размера 47.
(1 контейнер и 47 различных грузов)


Для этой задачи наилучшим значением заполненности является 0.773552.
(т.к. задача сгенерирована случайно, то нет решения, в котором контейнер полностью заполнен)

* Исследование_оптимального_ci *
Для каждого набора параметров алгоритм запускался 200 раз, после чего считалось среднее значение цф
для этого набора.
Наилучшие результаты алгоритм показывает при значениях Ci = 2.75 (коэффициент интенсивности мутации)
(либо можно указать, что при Ci на отрезке [2.7, 2.8])

* Исследование_оптимального_mp *
Для каждого набора параметров алгоритм запускался 150 раз, после чего считалось среднее значение цф
для этого набора
Наилучшие результаты алгоритм показывает при значениях Mp = 0.26 (вероятность мутации)
(либо можно указать, что при Mp на отрезке [0.23, 0.34])
